XAML-Perspective 1.0
date : 2010/01/30
Domain : 3D Visualization
Description :  XAML-Perspective is a subset of XAML for 3 dimensions using the Perspective API (available under MSPL license). The Perspective API provides an easier way to design 3D objects with XAML. It provides, notably, a complete toolset for changing camera position as well as a library of basic 2D and 3D objects (cubes, cylinders, etc.) The Perspective API was developed by Olivier Dewitt
