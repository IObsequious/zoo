TXL Base Grammar - ObjectPascal.Grm

A complete TXL base grammar for Borland Delphi Object Pascal
by Sylvain Leblanc, Royal Military College, Kingston, Canada,
summer 2002

Validated by parsing approx 50,000 lines of Object Pascal
code from the Canadian government.

Grammar extracted and added to the Grammar Zoo on 23 March 2013 by Vadim Zaytsev.
