ANTLR3 grammars for GraphViz DOT language

ANTLR3 grammars for GraphViz DOT language. Derived from specification on http://www.graphviz.org/doc/info/lang.html.

Grammars are part of CesTa project, http://cesta.sourceforge.net/. Parser produces AST, that can be used in tree parser to get model of given graph. Model is represented by class Graph. See API documentation for more details.

The files are:

  • Dot.g, the parser, that builds AST.
  • DotTree.g, the tree parser, that returns graph object.

Please report any problems to tsmolka(at)gmail(dot)com.
