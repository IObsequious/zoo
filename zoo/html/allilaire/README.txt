HTML 1.0
date : 2005/04/15
Domain : HTML
Description :  This basic metamodel describes HyperText Markup Language. HTML is the lingua franca for publishing hypertext on the World Wide Web. It is a non-proprietary format based upon SGML, and can be created and processed by a wide range of tools, from simple plain text editors - you type it in from scratch- to sophisticated WYSIWYG authoring tools. HTML uses tags such as &lt;h1&gt; and &lt;/h1&gt; to structure text into headings, paragraphs, lists, hypertext links etc.
