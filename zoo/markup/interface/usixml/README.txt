UsiXML Task Metamodel 1.0
date : 2008/11/15
Domain : User Interfaces Description
Description :  This metamodel describes the Usixml task organisation for describing User Interface at an high abstraction level. A task organization, describes an interactive application in term of user goals and actions. Its structure is a hierarchichal decomposition of user interactions.
