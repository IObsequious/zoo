Grammar
Here you can find the grammar for the Gosu programming language. The grammar below uses the following EBNF-style conventions:

[x] denotes zero or one occurrences of x.
{x} denotes zero or more occurrences of x.
x | y means one of either x or y.
For more information on this EBNF notation please refer to this wiki page: https://en.wikipedia.org/wiki/Wirth_syntax_notation
