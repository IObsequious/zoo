This directory contains “grammars” (in a broad sense) related to Java of unidentified version or of several versions.
For other Java grammars, see the sibling directories 'java-1', 'java-2', 'java-3', 'java-4', 'java-5', etc

Versions of Java programming language are:

1) “Java 1” = “JDK 1.0” - 1996, standardised by “The Java Language Specification”
	see http://java.sun.com/docs/books/jls/first_edition/html/index.html (taken offline)
	see http://java.sun.com/docs/books/jls/download/langspec-1.0.pdf (taken offline)
	see topics/grammars/java-1/jls-impl directory for grammar extraction from §19 of JLS
	see topics/grammars/java-1/jls-read directory for grammar extraction from JLS’ readable version

2) “Java 1.1” = “JDK 1.1” - 1997, changes covered in The Java Programming Language by Ken Arnold and James Gosling, Addison-Wesley, 1996, ISBN 0-201-63455-4.
	see http://java.sun.com/docs/books/jls/first_edition/html/1.1Update.html (informal)

3) “Java 2” = “J2SE 1.2” = “Playground” - 1998, standardised by “The Java Language Specification Second Edition”
	see http://java.sun.com/docs/books/jls/second_edition/html/j.title.doc.html (taken offline)
	see http://java.sun.com/docs/books/jls/download/langspec-2.0.pdf (taken offline)
	see http://java.sun.com/docs/books/jls/clarifications-2-2nd-ed.html (taken offline)
	see topics/grammars/java-2/jls-impl directory for grammar extraction from §18 of JLS2
	see topics/grammars/java-2/jls-read directory for grammar extraction from JLS2’s readable version

4) “Java 3” = “J2SE 1.3” = “Kestrel”, 2000, not standardised due to few language changes
	see http://java.sun.com/j2se/1.3/docs/relnotes/features.html

5) “Java 4” = “J2SE 1.4” = “Merlin”, 2002, not standardised
	see http://java.sun.com/j2se/1.4.2/docs/relnotes/features.html

6) “Java 5” = “J2SE 5.0” = “Tiger”, 2004, standardised by “The Java Language Specification Third Edition”
	see http://docs.oracle.com/javase/specs/jls/se5.0/html/j3TOC.html
	see http://docs.oracle.com/javase/specs/jls/se5.0/jls3.pdf
	see http://java.sun.com/j2se/1.5/docs/relnotes/features.html
	see http://jcp.org/aboutJava/communityprocess/maintenance/jsr924/index.html
	see topics/grammars/java-5/jls-impl directory for grammar extraction from §18 of JLS3
	see topics/grammars/java-5/jls-read directory for grammar extraction from JLS3’s readable version
	see topics/grammars/java-5/habelitz directory for grammar extraction from Dieter Habelitz’s version
	see topics/grammars/java-5/parr directory for grammar extraction from Terence Parr’s version
	see topics/grammars/java-5/stahl directory for grammar extraction from Michael Stahl’s version
	see topics/grammars/java-5/studman directory for grammar extraction from Michael Studman’s version

7) “Java 6” = “Java SE 6” = “Mustang”, 2006, not standardised (claimed to be mostly covered by JLS3)
	see http://jcp.org/aboutJava/communityprocess/maintenance/jsr924/index2.html
	see http://java.sun.com/javase/6/features.jsp
	see http://landonf.bikemonkey.org/static/soylatte/

8) “Java 7” = “Java SE 7” = “Dolphin”, 2011, standardised by “The Java Language Specification, Java SE 7 Edition”
	see http://docs.oracle.com/javase/specs/jls/se7/html/index.html
	see http://docs.oracle.com/javase/specs/jls/se7/jls7.pdf
	see http://openjdk.java.net/projects/jdk7/

9) “Java 8” = “Java SE 8”, under current (2013) development

10) “Java 9” = “Java SE 9”, planned for future development to be shipped in 2015
	see http://www.theregister.co.uk/2012/03/07/oracle_java_9_10_roadmap/
	see http://www.drdobbs.com/jvm/javaone-javafx-2-java-on-ios/231900029

11) “Java 10” = “Java SE 10”, planned for future development to be shipped in 2017
	see http://www.theregister.co.uk/2012/03/07/oracle_java_9_10_roadmap/
