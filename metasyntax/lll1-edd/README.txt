The “L^3_1”, which is the LLL from the “The Grammar Deployment Kit --- System Demonstration” by Jan Kort, Ralf Lämmel and Chris Verhoef.
(See directory ../lll-ldta for more information).

In this directory, you see the BGF grammar being extracted from Rascal code generated from the (textual) EDD file.
We include both the EDD and the generated Rascal file because running Rascal from the command line can be tricky.

For EDD see also:

	V. Zaytsev. BNF WAS HERE: What Have We Done About the Unnecessary Diversity of Notation for Syntactic Definitions.
	In Proceedings of the 27th ACM Symposium on Applied Computing (SAC’2012), Technical Track on Programming Languages.
